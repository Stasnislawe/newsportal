<h1 align="center">Новостной портал</h1>
<h2>Документация проекта через DeepWiki - https://deepwiki.com/Stasnislawe/newsportal/1-newsportal-overview</h2>
<h4 align="center">Находится в стадии разработки, не все реализации могут быть описаны</h4>
<p>Пользователь может зарегестрироваться на сайте, после чего, в личном кабинете заполнить форму, для того что бы получить роль автора.</p>
<img src="https://github.com/user-attachments/assets/8ce843b2-91be-437b-933c-0d558defa8ca">
<p>Роль автора даёт такие возможности как:</p>
<ul>
  <li>Создание/редактирование/удаление статей и новостей</li>
  <li>Подписка на категории постов(отправка на почту оповещения о новых постах на любимую категорию: при создании и еженедельно)</li>
</ul>
<p>После корректного заполнения формы, в личном кабинете появится информация о количестве постов, рейтинге, текущей роли, аватарки и краткого описания, и всё это можно редактировать</p>
<img src="https://github.com/user-attachments/assets/644c7aa5-06c1-41ba-becc-d84d4f78126f">
<p>Так же появются ссылки на создание новости и просмотра своих постов с последующим их редактированием/удалением, удалением коммента, который не понравился</p>
<img src="https://github.com/user-attachments/assets/bf133537-afec-4d25-9490-d13dc149b282">
<p>Зарегестрированный пользователь может только:</p>
<ul>
  <li>Просматривать новости</li>
  <li>Ставить лайки</li>
  <li>Писать комменты</li>
  <li>Удалять свои комменты</li>
  <li>Подписываться на категории</li>
</ul>

<h2 align="center">Подписка/Рассылки на почту</h2>
<p>Подписка на категории постов: если вам понравилась категория постов, то вы можете подписаться на нее => тем самым вы будете получать еженедельные рассылки на почту о новых постах за неделю на вашу любимую категорию, а так же при создании поста на подписанную категорию, автоматически вам на почту будет отправляться ссылка на этот пост. Если надоели рассылки, то в любой момент можно нажать кнопку "отписаться"</p>
<img style="width:50%;" src='https://github.com/user-attachments/assets/9fc64214-d35c-4008-8938-36a0a010b467'><img style="width:50%;" src='https://github.com/user-attachments/assets/f0114be1-748d-4f7d-acad-7020fb24febd'>
<p>Реализованно через Celery Redis Cloud</p>

<h2 align="center">Рейтинг</h2>
<div>
  <p>Рейтинг автора складывается из следующих параметров:</p>
    <ul>
      <li>Лайки за пост</li>
      <li>Лайки комментария оставленным другим пользователем под вашим постом</li>
      <li>Лайки за оставленный комментарий</li>
    </ul>
  <p>Формула рейтинга автора: <br> Рейтинг += (Лайки за пост) * 3 + (Лайки за комментарий оставленный др пользователем) + (Лайки за оставленный комментарий)</p>
</div>

<h2 align="">Реализация общения между пользователями</h2>
<p>Любой зарегестрированный пользователь может написать сообщение любому другому пользователю, даже автору, специально отведенный шаблон, в котором выведенены все пользователи</p>
<p>Если пользователь является автором, то у него будет отображаться его аватарка а так же псевдоним, рейтинг автора и количество его постов</p>
<img src="https://github.com/user-attachments/assets/fe049b0a-8fd6-43a1-9678-a5de7e19d33b">
<P>Если пользователь не открыл диалог с вами - то есть не прочитал, сообщение будет подсвечиваться, а на странице 'сообщения' непрочитанные сообщения также будут подсвечиваться галочками 1 и 2 соответственно</P>
